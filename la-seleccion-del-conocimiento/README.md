# La Selección del Conocimiento

Juego educativo para niños de escuela sobre informática básica.

## Idea del juego

El participante responde preguntas de opción múltiple. Cada respuesta correcta suma un jugador a la selección. El objetivo es completar una alineación de 11 jugadores y ganar el Mundial de la Informática.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Programación orientada a objetos mediante clases: `Pregunta`, `Jugador`, `Equipo` y `Juego`.

## Cómo abrirlo en VSCode

1. Descargar o clonar este repositorio.
2. Abrir la carpeta en Visual Studio Code.
3. Abrir el archivo `index.html` en el navegador.

También se puede usar la extensión Live Server de VSCode.

## Cómo publicarlo en GitHub Pages

1. Subir los archivos `index.html`, `style.css`, `script.js` y `README.md` a un repositorio de GitHub.
2. Entrar al repositorio en GitHub.
3. Ir a `Settings`.
4. Ir a `Pages`.
5. En `Build and deployment`, elegir `Deploy from a branch`.
6. En `Branch`, elegir `main` y la carpeta `/root`.
7. Guardar los cambios.
8. Esperar a que GitHub genere el enlace público del juego.
